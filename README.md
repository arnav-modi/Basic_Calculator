# Basic_Calculator
A simple command-line calculator built using Python that performs basic arithmetic operations and stores all previous calculations in a text file.
This project was created for practice and learning purposes
# Project Description
This calculator allows users to perform basic mathematical operations such as:
Addition
Subtraction
Multiplication
Division
Each calculation is saved in a history.txt file, which acts as a record of all previous calculations.
Users can view, clear, or edit the calculation history using simple commands.
# Working
~The user enters a calculation (example: 2 + 3)
~The program evaluates the expression
~The result is displayed on the screen
~The calculation is saved in history.txt
~Users can:
           View all previous calculations
           Clear the history file
          Continue performing new calculations
